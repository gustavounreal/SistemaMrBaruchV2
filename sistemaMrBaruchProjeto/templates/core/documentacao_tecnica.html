Configurações e Variáveis de Ambiente

Arquivo .env (Não versionado)
SECRET_KEY=your-secret-key-here
DEBUG=True
ALLOWED_HOSTS=localhost,127.0.0.1

# Database
DATABASE_URL=sqlite:///db.sqlite3

# Asaas Integration
ASAAS_API_KEY=your-asaas-key
ASAAS_API_URL=https://api.asaas.com/v3
ASAAS_WEBHOOK_TOKEN=your-webhook-token

Importante: Nunca versione o arquivo .env no Git. Use .env.example como template.

ConfiguracaoSistema (Banco de Dados)

Configurações dinâmicas armazenadas no banco, editáveis pelo admin panel

  • PIX_VALOR_LEVANTAMENTO - Valor da cobrança PIX
  • COMISSAO_PERCENTUAL - Percentual de comissão
  • LEAD_STATUS_INICIAL - Status padrão de novos leads
  • ASAAS_DIAS_VENCIMENTO - Dias até vencimento PIX

Deploy e Infraestrutura

1. Preparação
pip install -r requirements.txt
python manage.py collectstatic
python manage.py migrate
2. Servidor WSGI
gunicorn sistemaMrBaruchProjeto.wsgi:application
--bind 0.0.0.0:8000
--workers 4
3. Reverse Proxy
# Nginx
location / {
proxy_pass http://127.0.0.1:8000;
}
Checklist de Produção
  • DEBUG=False no .env
  • SECRET_KEY única e segura
  • ALLOWED_HOSTS configurado
  • Banco PostgreSQL configurado
  • Arquivos estáticos coletados (collectstatic)
  • HTTPS configurado (Let's Encrypt)
  • Backup automático do banco

Comandos Úteis

Desenvolvimento
# Rodar servidor de desenvolvimento
python manage.py runserver

# Criar migrações
python manage.py makemigrations

# Aplicar migrações
python manage.py migrate

# Criar superusuário
python manage.py createsuperuser
Manutenção
# Validar sistema
python manage.py check

# Shell interativo
python manage.py shell

# Dump do banco
python manage.py dumpdata > backup.json

# Restaurar banco
python manage.py loaddata backup.json
